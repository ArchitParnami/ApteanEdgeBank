APTEAN EDGE BANK

Steps to Build the application.

1.Build ApteanEdgeBankAPI
2.Build ApteanEdgeBankDB
3.Build ApteanEdgeBankUI


Application uses local SQL Server for database connection.
Please change the authentication method in SQL SERVER Services from given account to Local account.(from services.msc) 


On start
Application Username = archit
Application password = archit
